The MMO Server Project

The VaLNGOS (Valheim Large Network Game Object Server Suite) is a "from scratch" open-source implementation of the Valheim game server. The project's priorities include: enabling high performance 1000+ slot servers, ease of mod scripting and third party integration, un-modded client compatibility, and community driven leadership.

Planned Features
  • No Peer 2 Peer Networking. All game logic runs on the server.
  • Simple interface for modders. Write game logic in any programming language by interfacing with std streams.
  • Built in UDP frame analysis based anti-cheat.
  • Built in spam filter (naive-bayes)
  • Built in phasing load-balancer. Allows segmenting world zones to different threads/physical machines
  • Non-blocking io pipeline. Movement and world-entity UDP frames are never blocked.
  • Written in Rust. No garbage collection. Bare metal performance.
Low Priority
  • Mob pathing
  • Server-side physics

Progress

UPDATE #1: The VaLNGOS Core's feasibility has been proven. Connecting to servers and switching basic movement packets has been achieved. We are now working on FFI bindings for Steamworks and will proceed to implement a MVP.

The project is currently in the research phase. We are reverse engineering the game's networking code and planning a strategy for the server pipeline. Videos demonstrating basic behavior will come soon.
